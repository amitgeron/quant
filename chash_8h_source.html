53 /* .. c:type:: cf_chash_digest
54  * Hashing completion function type.
55  *
56  * Functions of this type complete a hashing operation,
57  * writing :c:member:`cf_chash.hashsz` bytes to `hash`.
58  *
59  * This function does not change `ctx` -- any padding which needs doing
60  * must be done seperately (in a copy of `ctx`, say).
61  *
62  * This means you can interlave `_update` and `_digest` calls to
63  * learn `H(A)` and `H(A || B)` without hashing `A` twice.
