Decodes dst_len bytes (if given, otherwise varint encoding is assumed) of network byte-order data starting at position pos of buffer buf (which has total length buf_len) info variable dst in host byte-order, using printf format string fmt to format the data for debug logging.

For varint decoding (dst_len is zero), dst must point to an uint64_t.

Parameters
dstDestination to decode into.
bufBuffer to decode from.
buf_lenBuffer length.
posBuffer position to start decoding from.
dst_lenLength to decode. Zero for varint decoding.
fmtPrintf format for debug logging.
Returns
Buffer offset of byte following the decoded data, or UINT16_MAX if the decoding failed.

Definition: marshall.c:128

Encodes src in host byte-order data into network byte-order at at position pos of buffer buf (which has total length buf_len), using printf format string fmt to format the data for debug logging.

src_len must be sizeof(src) for fixed-length encoding, or zero for "varint" encoding, in which case src must be uint64_t. In the latter case, if enc_len is zero, the "varint" encoding will use the minimal number of bytes necessary, otherwise, it will encode into enc_len bytes.

Parameters
bufBuffer to encode into.
buf_lenBuffer length.
posBuffer position to start encoding from.
srcSource data.
src_lenLength of src (zero means "varint" encoding).
enc_lenLength to encode (only affects "varint" encoding).
fmtPrintf format for debug logging.
Returns
Buffer offset of byte following the encoded data.
