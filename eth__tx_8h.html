Places an Ethernet frame into a TX ring.

The Ethernet frame is contained in the w_iov v, and will be placed into an available slot in a TX ring or - if all are full - dropped.

Parameters
wBackend engine.
vThe w_iov containing the Ethernet frame to transmit.
[in]lenThe length of the Ethernet payload contained in v.
Returns
True if the buffer was placed into a TX ring, false otherwise.
