73 /* .. c:type:: cf_aes_context
74  * This type represents an expanded AES key. Create one
75  * using :c:func:`cf_aes_init`, make use of one using
76  * :c:func:`cf_aes_encrypt` or :c:func:`cf_aes_decrypt`.
77  *
78  * The contents of this structure are equivalent to the
79  * original key material. You should clean the
80  * contents of this structure with :c:func:`cf_aes_finish`
81  * when you're done.
82  *
83  * .. c:member:: cf_aes_context.rounds
84  *
85  * Number of rounds to use, set by :c:func:`cf_aes_init`.
86  *
87  * This depends on the original key size, and will be
88  * :c:macro:`AES128_ROUNDS`, :c:macro:`AES192_ROUNDS` or
89  * :c:macro:`AES256_ROUNDS`.
90  *
91  * .. c:member:: cf_aes_context.ks
92  *
93  * Expanded key material. Filled in by :c:func:`cf_aes_init`.
94  */
